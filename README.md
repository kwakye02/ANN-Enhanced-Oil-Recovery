# ANN Enhanced Oil Recovery

Regression modelling project for enhanced oil recovery analysis using artificial neural networks. The project applies machine learning to petroleum engineering data to explore predictive modelling workflows.

## What This Covers

- Enhanced oil recovery data analysis
- Regression modelling
- Artificial neural network workflow
- Model training and evaluation
- Petroleum-focused machine learning practice

## How to Use

Open the notebook files in Jupyter Notebook, JupyterLab or Google Colab. Run the cells in sequence and review the model outputs and evaluation results.

## Tools

Python, Jupyter Notebook, pandas, NumPy, matplotlib and machine learning libraries.
